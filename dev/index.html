MarkovRandomFields.Models.IsingCoupling — Type
IsingCoupling

A type of Factor representing a factor in an Ising distribution. It involves $\pm 1$ variables $\boldsymbol{\sigma}_a=\{\sigma_1, \sigma_2, \ldots\}$. The factor evaluates to $\psi(\boldsymbol{\sigma}_a)=e^{\beta J \prod_{i\in a}\sigma_i}$. A particular case is the pairwise interaction where $a=\{i,j\}$ is a pair of vertices involved in an edge $(ij)$.

Fields

  • βJ: coupling strength.

MarkovRandomFields.Models.IsingField — Type
IsingField

A type of Factor representing a single-variable external field in an Ising distribution. The factor evaluates to $\psi(\sigma_i)=e^{\beta h \sigma_i}$.

Fields

  • βh: field strength.

MarkovRandomFields.Models.KSATClause — Type
KSATClause

A type of Factor representing a clause in a k-SAT formula. It involves $\{0,1\}$ variables $\boldsymbol{x}_a=\{x_1, x_2, \ldots, x_k\}$. The factor evaluates to $\psi_a(\boldsymbol{x}_a)=1 - \prod_{i\in a}\delta(x_i, J^a_{i})$.

Fields

  • J: a vector of booleans.
source
MarkovRandomFields.Models.PottsCoupling — Type
PottsCoupling

A type of Factor representing a factor in a Potts distribution. It involves k discrete variables $\boldsymbol{x}_a$, each taking a discrete value. The factor evaluates to $\psi(\boldsymbol{x}_a)=e^{\beta J(\boldsymbol{x}_a)}$. A particular case is the pairwise interaction where $a=\{i,j\}$ is a pair of vertices involved in an edge $(ij)$.

Fields

  • βJ: an Array with $k$ indices storing the (log) values of the factor for each input.

MarkovRandomFields.Models.PottsField — Type
PottsField

A type of Factor representing a single-variable external field in a Potts distribution. The factor evaluates to $\psi(x_i)=e^{\beta h(x_i})$.

Fields

  • βh: a vector storing the (log) values of the factor for each input.
